# Coursera Data Science Capstone Project

## Project Overview
This repository contains the work for the Coursera Data Science Specialization Capstone Project, conducted in collaboration with SwiftKey. The primary goal of this project is to build a predictive text model that suggests the next word based on previous input, similar to the technology used in mobile keyboards.

The project involves:
- Analyzing a large corpus of text documents.
- Cleaning and pre-processing text data for Natural Language Processing (NLP).
- Performing exploratory data analysis.
- Building and sampling from a predictive text model.
- Developing a Shiny application as a final data product.

## Dataset
The project uses the **Coursera-SwiftKey Dataset**, which contains text data from three different sources:
1. **Blogs**: Posts from various blogging platforms.
2. **News**: Articles from news websites.
3. **Twitter**: Short posts from Twitter.

The dataset can be downloaded from: [Coursera-SwiftKey Dataset](https://d396qusza40orc.cloudfront.net/dsscapstone/dataset/Coursera-SwiftKey.zip)

## Repository Structure
- `README.md`: Project overview and documentation.
- `TestProgram.R`: A collection of small R programs for practice, data downloading, and initial exploration.

## Getting Started
To run the scripts in this repository, you will need **R** and the following packages installed:
- `downloader`
- `plyr`
- `dplyr`
- `knitr`
- `readr`

You can install these packages using the following R command:
```R
install.packages(c("downloader", "plyr", "dplyr", "knitr", "readr"))
```
